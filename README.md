## VPS HOSTING

» Star and Fork the Repo.

» Open config.py

» Fill Bot Tokens, API, Owner ID, Sudo Users, Mongo Url

» Copy your repo link.

» Go to your VPS Terminal.

### VPS COMMANDS

» git clone repo_url sb

» cd sb

» pip install -r requirements.txt

» python3 alpha.py
